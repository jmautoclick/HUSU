# Política de Privacidad de Husu Habits

**Última actualización:** 10 de Mayo de 2026

Husu Habits es una aplicación diseñada para el seguimiento de hábitos de manera privada y local. Respetamos su privacidad y nos comprometemos a protegerla.

## 1. Recopilación y uso de datos
Husu Habits **NO** recopila, transmite, vende ni comparte ningún tipo de información personal, datos de uso ni estadísticas con servidores externos o bases de datos de terceros. Todo el progreso de tus hábitos, historial, configuraciones y datos de la aplicación se almacenan exclusivamente de forma **local** en el almacenamiento interno de tu dispositivo.

## 2. Servicios de Inteligencia Artificial (Coach IA)
La aplicación incluye una función opcional de "Coach IA". Para utilizar esta función, el usuario debe proporcionar su propia clave de API (API Key) de Google Gemini. Las consultas realizadas al Coach se envían directamente desde el dispositivo del usuario a los servidores de Google Gemini. Husu Habits no almacena estas interacciones en ningún servidor propio. El uso de esta función está sujeto a las políticas de privacidad de Google.

## 3. Permisos requeridos
La aplicación puede solicitar permiso para enviar **Notificaciones Locales** (para los recordatorios de hábitos). Estos recordatorios se programan en tu propio teléfono de manera local y no requieren conexión a internet para funcionar.

## 4. Almacenamiento y copias de seguridad
Los datos generados en la aplicación pertenecen 100% al usuario. La aplicación ofrece una función de copia de seguridad que permite al usuario exportar manualmente sus datos en un archivo dentro de su dispositivo e importarlos posteriormente.
